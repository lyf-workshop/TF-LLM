# Youtu-Agent KORGym 实验版本

> 本项目基于 [Tencent Youtu-Agent](https://github.com/TencentCloudADP/youtu-agent) 开发，专注于 KORGym 游戏环境的实验和研究。

## 🎯 本版本特色

相比原版 Youtu-Agent，本版本增加了：

### 📚 完整的 KORGym 文档体系
- **Wordle 完整指南** - 多轮交互猜单词游戏
- **Word Puzzle 完整指南** - 单轮填字游戏
- **Alphabetical Sorting 完整指南** - 字母排序游戏
- **系统化问题排查指南** - 覆盖 15+ 常见问题
- **快速命令参考** - 所有常用命令速查

### 🎮 KORGym 游戏实验配置
- 三个游戏的完整配置文件（Agent、Practice、Eval）
- 配置文件模板（快速为新游戏创建配置）
- 多轮游戏评估支持（Wordle 等）
- 层次经验学习优化

### 🔧 优化和工具
- Wordle Agent 提示词优化（精简 26%）
- 重试机制详解和配置指南
- 数据集准备脚本
- 结果分析工具
- 文档清理脚本
